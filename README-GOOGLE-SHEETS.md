# Инструкция по обновлению раздела "Выполненные заказы"

## Как это работает

Раздел "Выполненные заказы" на сайте автоматически обновляется из Google Таблицы при сборке сайта. Это позволяет легко добавлять и редактировать примеры выполненных работ без изменения кода.

## Ссылка на таблицу

Google Таблица для управления карточками: 
https://docs.google.com/spreadsheets/d/e/2PACX-1vRLMI4GeevGP-r_mrPLntMsbXJ0mFUGfDtMKCjFuOSUhNhy-UatGDpSsMDebHeBMwyTnRG5K_d-POIB/pub?gid=217344138&single=true&output=csv

## Структура таблицы

Таблица должна содержать следующие колонки (в указанном порядке):

| Колонка    | Описание                                  | Пример                           |
|------------|-------------------------------------------|----------------------------------|
| title      | Модель автомобиля                         | Toyota Harrier                   |
| year       | Год выпуска                               | 2021                             |
| client     | Информация о клиенте                      | клиентке из Улан-Удэ            |
| engine     | Объем двигателя                           | 2.5л                             |
| mileage    | Пробег                                    | 33.000км                         |
| condition  | Состояние автомобиля                      | аукционная оценка 4,5 балла     |
| price      | Цена                                      | 3.050.000                        |
| priceNote  | Примечание к цене                         | под ключ                         |
| country    | Страна происхождения                      | Япония                           |
| date       | Дата завершения заказа                    | Январь 2025                      |
| image      | Путь к изображению                        | /bgauto/images/case-1.jpg       |

## Как добавить новый пример

1. Откройте Google Таблицу (ссылка выше)
2. Добавьте новую строку с данными о выполненном заказе
3. Заполните все колонки согласно структуре
4. Убедитесь, что изображение загружено в папку `/public/bgauto/images/`
5. Запустите обновление сайта (см. раздел ниже)

## Как обновить существующий пример

1. Откройте Google Таблицу
2. Найдите нужную строку
3. Внесите изменения
4. Запустите обновление сайта

## Как обновить сайт

После внесения изменений в таблицу:

1. Перейдите на GitHub: https://github.com/[ваш-username]/[ваш-репозиторий]
2. Откройте вкладку "Actions"
3. Выберите workflow "Deploy to GitHub Pages"
4. Нажмите "Run workflow" → "Run workflow"
5. Дождитесь завершения сборки (обычно 2-5 минут)
6. Сайт обновится автоматически

## Важные примечания

- Таблица должна быть опубликована как CSV (уже настроено)
- Не изменяйте названия колонок в первой строке
- Не используйте запятые в данных (или используйте кавычки)
- Изображения должны быть загружены на сервер до добавления в таблицу
- Рекомендуемый размер изображений: соотношение сторон 9:16 (вертикальные)
