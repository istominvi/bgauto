# Инструкция по обновлению раздела "Выполненные заказы"

## Как это работает

Раздел "Выполненные заказы" на сайте автоматически обновляется из Google Таблицы при сборке сайта через GitHub Actions. Это позволяет легко добавлять и редактировать примеры выполненных работ.

## Ссылка на таблицу

Google Таблица для управления карточками: 
https://docs.google.com/spreadsheets/d/e/2PACX-1vRLMI4GeevGP-r_mrPLntMsbXJ0mFUGfDtMKCjFuOSUhNhy-UatGDpSsMDebHeBMwyTnRG5K_d-POIB/pub?gid=217344138&single=true&output=csv

## Структура таблицы

Таблица должна содержать следующие колонки (в первой строке):

| Колонка      | Описание                                  | Пример                           |
|--------------|-------------------------------------------|----------------------------------|
| Автомобиль   | Модель и год через запятую                | Toyota Harrier, 2021             |
| Кому         | Информация о клиенте                      | Алексею из Улан-Удэ             |
| Двигатель    | Объем двигателя                           | 2.5 л                            |
| Пробег       | Пробег                                    | 33,000 км                        |
| Состояние    | Состояние автомобиля                      | Аукционная оценка 4,5 балла      |
| Цена         | Цена (можно с ₽)                          | 3,050,000 ₽                      |
| Страна       | Страна происхождения                      | Япония                           |
| Дата         | Дата завершения заказа                    | Январь 2025                      |
| Фото         | Ссылка на фото (Google Drive)             | https://drive.google.com/...     |

*Примечание: Скрипт также поддерживает английские названия колонок (title, year, client, engine, mileage, condition, price, country, date, image).*

## Как добавить новый пример

1. Откройте Google Таблицу (ссылка выше).
2. Добавьте новую строку с данными о выполненном заказе.
3. Заполните колонки согласно структуре.
4. Для фотографий используйте прямые ссылки или ссылки на Google Drive (скрипт автоматически их обработает).
5. Запустите обновление сайта.

## Как обновить сайт

После внесения изменений в таблицу:

1. Перейдите на GitHub в раздел **Actions**.
2. Выберите workflow **Deploy to GitHub Pages**.
3. Нажмите **Run workflow** → **Run workflow**.
4. Сайт обновится автоматически в течение нескольких минут.

## Важные примечания

- Не изменяйте названия колонок в первой строке, если не планируете обновлять скрипт `scripts/fetch-cases.js`.
- Если вы используете Google Drive для фото, убедитесь, что доступ к файлу открыт "Всем, у кого есть ссылка".
- Рекомендуется использовать вертикальные фото (9:16) для лучшего отображения в карточках.

# Инструкция по обновлению раздела "Отзывы"

## Как это работает

Раздел "Отзывы" также автоматически обновляется из Google Таблицы.

## Ссылка на таблицу

Google Таблица для отзывов:
https://docs.google.com/spreadsheets/d/e/2PACX-1vRLMI4GeevGP-r_mrPLntMsbXJ0mFUGfDtMKCjFuOSUhNhy-UatGDpSsMDebHeBMwyTnRG5K_d-POIB/pub?gid=0&single=true&output=csv

## Структура таблицы (Отзывы)

| Колонка      | Описание                                  |
|--------------|-------------------------------------------|
| Отзыв        | Текст отзыва                              |
| Имя          | Имя клиента                               |
| Страна       | Страна (Япония, Корея, Китай)             |
| Автомобиль   | Модель автомобиля                         |
| Экономия     | Сумма экономии (если есть)                |

Обновление происходит так же, как и для раздела "Выполненные заказы", через запуск GitHub Actions.
